# 📝 ToDo App

A simple, stylish, and responsive ToDo application built with HTML, CSS, and JavaScript.

## 🚀 Features

- **Add Tasks:** Quickly add new tasks to your list.
- **Mark as Complete:** Check off tasks as you finish them.
- **Delete Tasks:** Remove tasks you no longer need.
- **Persistent Storage:** Your tasks are saved in your browser (localStorage).
- **Responsive Design:** Looks great on both desktop and mobile devices.

## 🛠️ Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/vertex616/Todo-App.git
   ```
2. **Open the project folder:**
   ```
   cd Todo-App
   ```
3. **Open `index.html` in your browser**  
   or use [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) for live reloading.

## 📱 Mobile Preview

To preview on your phone, run a local server (like Live Server) and open the server URL on your mobile device.

## ✨ Customization

- Edit `style.css` to change the look and feel.
